Understanding changes in air quality is important for public health. The report for this R code aims to investigate changes in PM2.5 air quality from 2022 and 2024. This is done through 4 main visualisation designs using ggplot. The overall findings of this report is PM2.5 has decreased in Sheffield from 2022 to 2024. This report was limited to the quanity of research but future work could add extra years or compare different cities for stronger analysis.
